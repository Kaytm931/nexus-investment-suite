## Technische Details

### Tavily Vorher/Nachher

| Phase | Vorher | Nachher |
|---|---|---|
| Initial (gather_qualitative) | 2 Suchen × advanced × 4 Ergebnisse = ~8 Credits | 1 Suche × basic × 3 Ergebnisse = ~1 Credit |
| Agentic Loop (search_fn) | 2 Suchen × advanced × 2 Ergebnisse = ~8 Credits | 2 Suchen × basic × 1 Ergebnis = ~2 Credits |
| Gesamt pro Analyse | ~16 Credits | ~3 Credits (~80% Ersparnis) |

### Company Name Flow

```
yFinance.get_ticker_data("ORC.DE")
  → yf_data["name"] = "Oracle Corporation"
  → company_name = "Oracle Corporation"

gather_ticker_qualitative(ticker="ORC.DE", company_name="Oracle Corporation")
  → query = "Oracle Corporation insider buying competitive advantage moat risks 2025"
  → Tavily findet Oracle, nicht Orchid Island ✓

user_message = "ANALYSE-AUFTRAG: ORC.DE — Oracle Corporation
⚠️ UNTERNEHMEN: Oracle Corporation (Ticker: ORC.DE)..."
  → LLM weiß welche Firma gemeint ist ✓
```
