## Skill Overview

A practical skill for designing and implementing product‑grade web motion systems used by teams at Stripe, Linear, Apple, and Vercel. It codifies motion principles, recommended easing and duration defaults, choreography patterns (entrance, exit, stagger, hero transitions), and common scroll/hover interaction recipes. The skill includes performance best practices (compositor-safe transforms, requestAnimationFrame, layer management), accessibility guidance (reduced-motion fallbacks, prefers-reduced-motion strategies), and implementation advice across CSS, animation libraries, and component libraries. Use it when building landing pages, dashboards, micro‑interactions, and page transitions to ensure consistent, performant, and accessible motion. Core advantages: faster designer‑developer handoff, predictable motion tokens, better perceived performance, and reduced cognitive load for users.
