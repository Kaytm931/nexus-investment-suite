## Skill Overview

React-first animation Skill that delivers Awwwards/FWA-level motion using GSAP (useGSAP), Framer Motion, Anime.js and Lenis for buttery-smooth 60fps experiences. Provides reusable hooks and patterns for smooth scroll, ScrollTrigger, Lenis integration, magnetic cursor effects, reveal/stagger animations, horizontal scroll and pinned sections, page transitions, micro-interactions, and custom cursors. Built for complex scenarios including useScroll/useTransform workflows, Three.js/WebGL integration, algorithmic and generative art (fractals, L‑systems, flow fields, tessellations, Penrose tiles), kinetic typography, glitch effects and morphing text. Emphasizes proper React cleanup, SSR-friendly hooks, performance-conscious batching and requestAnimationFrame strategies. Use this Skill for premium portfolios, interactive landing pages, creative showcases and anything that requires precise timing, high frame-rate fidelity and production-ready animation architecture.
