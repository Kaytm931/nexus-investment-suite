## Skill Overview

GSAP (GreenSock Animation Platform) is a high-performance JavaScript animation library for building complex, production-quality web animations. It offers imperative APIs (gsap.to, gsap.from, gsap.set), timeline sequencing with precise playback controls (play, pause, reverse, seek), and staggering/easing for polished choreography. With plugins like ScrollTrigger you can create scroll-driven effects, pinned sections, scrubbed horizontal scrollers, and enter/leave toggles. GSAP also supports SVG morphing, rich text effects, and physics-based motion for natural interactions. Use it for landing pages, product showcases, interactive UI transitions, animated content, and micro-interactions where CSS cannot provide the needed control or performance. Core advantages include frame-perfect timing, broad browser support, minimal layout thrashing, and an extensible plugin ecosystem used by major brands.
